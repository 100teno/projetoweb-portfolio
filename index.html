Projetos Recentes

Landing Page Responsiva

Desenvolvimento exemplo de uma landing page simples totalmente responsiva com flex boxes, grids, imagens na galeria com animações, menu fixo, interativo e objetivo, botão para retornar ao topo e menu expansível responsivo utilizando apenas HTML e CSS (JavaScript é utilizado na maioria dos desenvolvimentos, essa é uma exceção onde ele não foi aplicado).

Aplicativo de Clima

Um aplicativo de Clima em tempo real desenvolvido em HTML, CSS e JavaScript, utilizando a aplicação de uma API Pública para as informações atualizadas.
A API utilizada é de uso gratuito e fornece todos os dados de condições climáticas em tempo real de todas as cidades do mundo.
O usuário digita qualquer cidade do mundo e os dados climáticos em tempo real aparecem na tela.

Calculadora de porcentagem

Este é um aplicativo simples de uma calculadora de porcentagem, desenvolvido utilizando HTML, CSS e JavaScript, aplicando lógica básica de programação com o JS para calcular quanto é a porcentagem de um número. O usuário deve inserir dois valores, a porcentagem que deseja descobrir o valor e o número, e o aplicativo irá calcular.

Calculadora de IMC
(Índice de massa corporal)

Um app simples para calcular seu IMC utilizando lógica básica de JavaScript coletando dois dados do usuário, sendo eles peso e altura, e calculando seu IMC tendo como base a formula para calculo de IMC.
A calculadora te retona o seu IMC e em qual classificação ele se encontra.

Lista de Tarefas

Uma simples lista de tarefas utilizando lógica com JavaScript, com botões para adicionar e apagar tarefas. O app salva os dados da lista no navegador local em uma JSON string, para um próximo uso,
onde as tarefas salvas não serão perdidas.
O foco do projeto foi a lógica e não estilização.
